# Beta Divergence Loss - PyTorch Implementation

## Overview

This repository contains code of a PyTorch implementation of the beta divergence loss.


## Dependencies

This package is written in Python, and requires Python (with recommended version >= 3.9) to run. This package relies on the following libraries and version numbers:

* [Python](https://www.python.org/) >= 3.9
* [NumPy](https://numpy.org/) >= 1.22.0
* [SciPy](https://www.scipy.org/) >= 1.7.3


## Installation

To install the latest stable release, use [pip](https://pip.pypa.io/en/stable/). Use the following command to install:

    $ pip install pytorch-beta-divergence
